# Privacidade

PulseMac e 100% offline.

- Nenhuma chamada de rede
- Sem telemetria, analytics ou crash reporting
- Sem remote config
- Dados ficam somente no dispositivo
- Alertas e historico sao locais
- App Sandbox desativado para permitir leitura de processos (somente local)

Nao solicitamos permissao de rede e nao enviamos dados para fora do dispositivo.
